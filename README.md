# A New Pushdown Demo for AD
## Clone & Install
### 1.
```
git clone https://github.com/FTChinese/ad-pushdownDemo.git 
```
### 2.
```
npm install
```
### 3.
```
npm rm gulp
```
### 4.
```
npm install "github:gulpjs/gulp#4.0"
```
## Local Test
by "dev" mode
```
gulp serve

```
## Online Test 
by "prod" mode
```
gulp deploy
```
